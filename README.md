# dotfiles

Personal configuration files for my Linux environment. Version controlled and publicly mirrored on GitHub and Codeberg.

The goal is a stateless, reproducible environment — any machine running Fedora or Rocky Linux can be brought to my exact configuration from a single clone. Documents how I work, how my tools are configured, and evolves alongside my Linux studies.

## Contents

| File | Description |
|------|-------------|
| `.vimrc` | Vim configuration — editing, indentation, search, appearance |

## Usage

Clone the repo:

```bash
git clone https://github.com/fionnlinux/dotfiles.git
```

Symlink or copy the files you want:

```bash
cp dotfiles/.vimrc ~/.vimrc
```

## Environment

- **Daily driver:** Fedora KDE
- **Server:** Rocky Linux
- **Editor:** Vim
